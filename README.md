# React Big Calendar with Bar Graph

## Overview
This project is a **React-based calendar application** integrated with **React Big Calendar** and **Recharts** to visualize user data on specific dates. Users can view data on a daily, weekly, or monthly basis. The app highlights dates with available data and displays a popup with a **bar graph** when a highlighted date is clicked. If no data exists for a selected date, a warning message is shown.  

The project uses **Redux** for state management and **React Toastify** for notifications.

---

## Features
- Display calendar with **daily, weekly, and monthly views**.
- Highlight dates with available data.
- Highlight the selected date.
- On clicking a highlighted date:
  - Show a popup with the selected date.
  - Display a **bar graph** representing the data for that date.
- Show a warning popup if no data exists for the selected date.
- Uses **dummy JSON data** for demonstration.
- Fully functional with **Redux state management**.
- Smooth operation on all major operating systems.

---

## Tech Stack / Dependencies
- **React** (Frontend framework)
- **Redux** (State management)
- **React Big Calendar** (Calendar UI)
- **Recharts** (Bar chart visualization)
- **React Toastify** (Notifications / Alerts)
- **date-fns** (Date utilities)
- **CSS / Tailwind / SCSS** (Styling, optional)
